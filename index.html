¿Qué es el Método Simplex?

El método Simplex es una técnica matemática utilizada para optimizar y resolver problemas de programación lineal, es decir, problemas que buscan maximizar o minimizar una función objetivo lineal sujeta a una serie de restricciones lineales. Se usa comúnmente en campos como la logística, la economía, la ingeniería y la administración de operaciones.

¿Para qué sirve?

El método Simplex es utilizado principalmente en el campo de la investigación operativa para optimizar el uso de recursos limitados, tales como materiales, tiempo, o dinero, en la toma de decisiones empresariales.

  • Maximización de beneficios.
  • Minimización de costos.
  • Asignación óptima de recursos.

Características clave del método Simplex

  1. Función Objetivo

    2. Es una expresión lineal que representa la meta del problema, como maximizar las ganancias o minimizar los costos.
    Por ejemplo: Maximizar Z=3x1+2x2 ​
    Donde x1, x2 son las variables de decisión.

  2. Restricciones

    3. Son un conjunto de ecuaciones o desigualdades lineales que limitan los valores que las variables de decisión pueden tomar.
    Por ejemplo:
    2x1 + 3x2 ≤ 6
    x1 + x2 ≤ 4

  3. Variables no negativas

    4. En programación lineal, las variables de decisión suelen estar limitadas a valores no negativos (x1 ≥ 0, x2 ≥ 0).

  4. Tabla Simplex

    5. Es una herramienta visual que representa el problema de programación lineal en forma tabular, facilitando la resolución del problema. Por ejemplo:

    Variable
    Básica    		 Z x1 x2 S1 S2 R
    F0 Z 1 -3 -2 0 0 0
    F1 S1 0 2 3 1 0 6
    F2 S2 0 1 1 0 1 4

Reglas del Método Simplex

A continuación se presentan las reglas que debes seguir al aplicar el método Simplex:

  • La función objetivo debe ser de maximización o minimización, pero para esta calculadora las restricciones deben ser con menor o igual (≤).
  • Las restricciones deben estar en forma de ecuaciones lineales, igualadas a un valor constante.
  • Se agregarán variables de holgura teniendo en cuenta el número de restricciones, para convertir desigualdades en igualdades.
  • Durante el proceso, se identificará la columna y la fila pivote correctamente para cada iteración, teniendo en cuenta si se trata de maximización o minimización.

Comprobación de estado. Para comprobar el estado de la solución, se verificará en la fila cero (F0: fila o renglón de la tabla donde se ubica la función objetivo Z). Para Maximización se observa si todos los coeficientes son ceros o valores POSITIVOS. Para Minimización se observa si todos los coeficientes son ceros o valores NEGATIVOS.

Nueva Variable Básica (NVB). En el proceso del método simplex las variables básicas iniciales son las variables de holgura (Si). Para Maximización La nueva variable básica (NVB) es aquella que en la fila cero tenga en valor más alto pero NEGATIVO. Para Minimización La nueva variable básica (NVB) es aquella que en la fila cero tenga en valor más alto pero POSITIVO.

Variable Básica de Salida (VBS). Se divide cada termino independiente (Columna R en la tabla) sobre los coeficientes de la NVB y se selecciona el menor de ellos (los cocientes negativos o indeterminados no se tienen en cuenta).

Para realizar el proceso del método simplex, después de marcar la columna de la NVB y la fila de la VBS se determina el pivote y hacen las respectivas transformaciones (en el fondo es utilizar las operaciones elementales aplicada a las filas de la matriz)

Algoritmo del Método Simplex

Un Modelo de Programación Lineal (MPL) se transforma a su forma estándar y se construye una matriz con los coeficientes del sistema de ecuaciones formado por los coeficientes de la función objetivo y los coeficientes de las restricciones. La función objetivo puede ser de maximización o minimización, y se expresa de la siguiente manera:

Maximizar/Minimizar Z = c1x1 + c2x2 + ... + cnxn

Sujeto a restricciones:

  • a11x1 + a12x2 + ... + a1nxn ≤ b1
  • a21x1 + a22x2 + ... + a2nxn ≤ b2
  • ...
  • am1x1 + am2x2 + ... + amnxn ≤ bm

Pasos para aplicar el método Simplex:

  1. Forma estándar: El problema se reformula para tener ecuaciones de igualdad, introduciendo variables de holgura para transformar las desigualdades.
  2. Tabla Simplex: Se construye una tabla que representa el problema en una forma matricial. Cada columna representa una variable, y cada fila una ecuación.
  3. Pivoteo: El método selecciona una variable "entrante" que mejora la solución y una variable "saliente" que abandona la base, recalculando la tabla iterativamente.
  4. Iteraciones: Se repite el proceso de pivotar hasta alcanzar la solución óptima, o se determina que el problema no tiene solución factible o está sin límite.
